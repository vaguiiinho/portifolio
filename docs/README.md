# Índice de Documentos

Esta pasta separa os documentos por finalidade.

## Mapa dos arquivos

| Arquivo | Tipo | Uso |
| --- | --- | --- |
| `prd-backend-api.md` | PRD backend | API, domínio, Prisma, módulos e regras de negócio do backend |
| `prd-frontend-portfolio.md` | PRD frontend | Conversão, conteúdo, páginas e estrutura do portfólio |
| `prd-frontend-arquitetura.md` | PRD frontend | Refino de arquitetura, organização e separação de responsabilidades no frontend |
| `prd-auth-permissoes.md` | PRD transversal | Autenticação, papéis e permissões |

## Convenção

- `prd-` indica documento de produto/execução.
- O sufixo descreve a área principal.
- Quando o documento afetar mais de uma área, ele deve ficar aqui e entrar neste índice.
